## Rodando a aplicação 🎲

```bash
# Clone este repositório
$ git clone <https://github.com/uesleicarvalhoo/vision-manager-portal>

# Acesse a pasta do projeto no terminal
$ cd vision-manager-portal

# Instale as dependencias

$ npm install

# Execute o ambiente de dev

$ npm run dev
# Você pode acessar o site através da url <http://localhost:3000/>
```
